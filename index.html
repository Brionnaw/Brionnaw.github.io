About Me

I am a highly productive, self-motivated and adaptable WordPress & front-end developer who has experience in JavaScript programming and WordPress based in the Sacramento Area.

Working in tech was something that I fell into; it wasn’t exactly a conscious decision. As a teenager, I ended up teaching myself how to code and fell in love with it. I decided to explore more into programming and immerse myself into learning web development. From there, it turned into a career. I found very quickly that I had a knack for web development and I couldn’t get enough.

In spite of the very few women of color in technology, it makes me become a better developer and critical thinker every day. I enjoy creating clean, intuitive and responsive applications to provide the best user experiences.

I create the best user experiences. I have experience building custom responsive websites with HTML5, CSS3, jQuery, JavaScript and PHP, in addition to being able to develop websites with Git, JavaScript frameworks, and WordPress from start to finish.

I get the job done. Aside from programming, you can catch me vlogging, taking selfies, or enduring hard cardio to better my work-life balance.
